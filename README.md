# FINTRACE AI - Financial Forensics Engine

A professional landing page for a financial investigation tool used by government agencies and financial institutions.

## Features

- Dark/Light mode toggle
- Responsive design
- Professional forensic UI style
- Built with React and Tailwind CSS

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Theme Toggle

The application supports both dark and light modes. Use the theme toggle button in the navbar to switch between themes.
